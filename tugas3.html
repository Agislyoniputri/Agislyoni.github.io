Constructor dan Enkapsulasi di Dart

Agis Lyoni Putri | NIM 251410022 | Kelas SI2A

Pengertian Constructor

Constructor adalah method khusus dalam sebuah class yang digunakan untuk membuat dan menginisialisasi objek. Constructor akan otomatis dipanggil ketika objek dibuat. Tujuannya adalah memberikan nilai awal pada atribut yang ada di dalam class.

Jenis-jenis Constructor

1. Default Constructor adalah constructor dasar yang digunakan untuk membuat objek.

2. Named Constructor adalah constructor yang memiliki nama tambahan sehingga dalam satu class bisa memiliki lebih dari satu constructor.

3. Constant Constructor digunakan untuk membuat objek yang bersifat tetap atau immutable.

4. Constructor dengan Initializer List digunakan untuk menginisialisasi nilai variabel sebelum constructor dijalankan.

Enkapsulasi

Enkapsulasi adalah konsep dalam pemrograman berorientasi objek yang digunakan untuk melindungi data agar tidak dapat diakses langsung dari luar class. Data biasanya dibuat private dengan menggunakan tanda underscore (_) pada nama variabel.

Public dan Private

Variabel public dapat diakses dari luar class, sedangkan variabel private hanya dapat diakses di dalam class atau file yang sama. Di Dart, variabel private ditandai dengan tanda underscore (_).

Getter dan Setter

Getter digunakan untuk mengambil nilai dari variabel private, sedangkan setter digunakan untuk mengubah nilai variabel tersebut dengan aturan atau validasi tertentu. Dengan menggunakan getter dan setter, data dapat lebih aman dan terkontrol.
