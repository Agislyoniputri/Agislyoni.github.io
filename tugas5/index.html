pengertian

Multiple inheritance adalah konsep di mana sebuah class dapat mewarisi sifat (atribut & method) dari lebih dari satu class induk (parent class).

Class modifiers adalah kata kunci yang digunakan untuk mengubah perilaku atau sifat dari sebuah class, seperti `abstract`, `interface`, `sealed`, dan `base`.

Studi Kasus: mixin dengan state

Mixin digunakan untuk menambahkan kemampuan ke dalam sebuah class tanpa harus menggunakan banyak pewarisan.Contohnya, class Bird bisa terbang, sedangkan Duck bisa terbang dan juga berenang karena menggunakan dua mixin yang berbeda.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

mixin CanFly {
  bool _isFlying = false;
  bool get isFlying => _isFlying;

  void fly() {
    if (!_isFlying) {
      _isFlying = true;
      print('Started flying!');
    } else {
      print('Already flying!');
    }
  }

  void land() {
    if (_isFlying) {
      _isFlying = false;
      print('Landed!');
    } else {
      print('Already on the ground!');
    }
  }
}

mixin CanSwim {
  bool _isSwimming = false;
  bool get isSwimming => _isSwimming;

  void swim() {
    if (!_isSwimming) {
      _isSwimming = true;
      print('Started swimming!');
    } else {
      print('Already swimming!');
    }
  }

  void stopSwim() {
    if (_isSwimming) {
      _isSwimming = false;
      print('Stop swimming!');
    } else {
      print('Already on stop swimming!');
    }
  }
}

class Bird with CanFly {}
class Duck extends Bird with CanSwim {}
class Fish with CanSwim {}
class Airplane with CanFly {}

void main() {
  Bird bird = Bird();
  bird.fly();
  bird.fly();
  bird.land();
  bird.land();

  Duck duck = Duck();
  duck.fly();
  duck.swim();
  duck.swim();
  duck.stopSwim();
}

Sealed Class

Pada bagian ini, mixin tidak hanya berisi fungsi, tetapi juga memiliki state (kondisi) seperti: sedang terbang atau tidak sedang berenang atau tidak State ini disimpan dalam variabel (misalnya _isFlying, _isSwimming). Fungsi fly() dan swim() tidak hanya menjalankan aksi, tetapi juga memeriksa dan mengubah state tersebut. Ini membuat mixin lebih canggih karena bisa menyimpan informasi tentang kondisi objek, bukan hanya menambahkan fungsi.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void handleResponse(ApiResponse response) {
  if (response is Success) {
    print("Success: ${response.message}");
  } else if (response is Error) {
    print("Error: ${response.errorMessage}");
  }
}

void main() {
  handleResponse(Success("Berhasil"));
  handleResponse(Error("Gagal"));
}

Pengantar Abstraksi

Pengantar abstraksi adalah penjelasan awal tentang konsep abstraksi dalam OOP, yaitu proses menyederhanakan sistem dengan menyembunyikan detail yang tidak penting dan hanya menampilkan bagian utama yang dibutuhkan.

Abstract Class

Abstract class adalah kelas yang tidak bisa diinstansiasi dan biasanya digunakan sebagai blueprint untuk kelas lain. Abstract class dapat memiliki method abstract (tanpa implementasi) yang harus diimplementasikan oleh subclass.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

abstract class Kendaraan {
  String? merk;

  void start();

  void stop() {
    print('Kendaraan berhenti');
  }
}

class Mobil extends Kendaraan {
  @override
  void start() {
    print('Mobil $merk menyala');
  }
}

void main() {
  Mobil m = Mobil();
  m.merk = 'Toyota';

  m.start();
  m.stop();
}

Static member diakses langsung melalui class name.Properti dan method static sangat berguna untuk membuat utility class atau menyimpan konstanta yang terkait dengan class, sehingga dapat diakses langsung tanpa perlu membuat objek.

Studi Kasus: Sistem Bentuk Geometri

abstract class Bentuk {
  double hitungLuas();
  double hitungKeliling();

  void info() {
    print('Luas: ${hitungLuas()}, Keliling: ${hitungKeliling()}');
  }
}

class Persegi implements Bentuk {
  double sisi;
  Persegi(this.sisi);

  @override
  double hitungLuas() => sisi * sisi;

  @override
  double hitungKeliling() => 4 * sisi;

  @override
  void info() {
    print(
        'Persegi sisi $sisi, luas ${hitungLuas()}, keliling ${hitungKeliling()}');
  }
}

class Lingkaran implements Bentuk {
  double radius;
  Lingkaran(this.radius);

  @override
  double hitungLuas() => 3.14 * radius * radius;

  @override
  double hitungKeliling() => 2 * 3.14 * radius;

  @override
  void info() {
    print(
        'Lingkaran radius $radius, luas ${hitungLuas()}, keliling ${hitungKeliling()}');
  }
}

void main() {
  Persegi p = Persegi(4);
  Lingkaran l = Lingkaran(7);

  p.info();
  l.info();

kesimpulan

    Polimorfisme dan abstraksi adalah konsep penting dalam OOP yang membuat program lebih rapi, fleksibel, dan mudah dikembangkan.Polimorfisme memungkinkan satu class induk digunakan oleh banyak class anak dengan fungsi yang berbeda, sedangkan abstraksi menyembunyikan detail program dan hanya menampilkan fungsi utama.Dengan menggunakan abstract class, interface, dan static method, program menjadi lebih terstruktur, mudah dipahami, dan bisa digunakan kembali di berbagai kebutuhan.

Bonus 1

Latihan ini menggabungkan Abstract Class dan Polimorfisme. Abstract class Pekerja berfungsi sebagai template yang mendefinisikan kontrak (method bekerja()) dan memberikan implementasi default (method istirahat()).

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

// Abstract class Pekerja
abstract class Pekerja {
  void bekerja(); // method abstract
  
  void istirahat() { // method concrete
    print('Sedang istirahat...');
  }
}

// Class Programmer
class Programmer implements Pekerja {
  String nama;
  Programmer(this.nama);

  @override
  void bekerja() {
    print('$nama (Programmer) sedang menulis kode Dart');
  }

  @override
  void istirahat() {
    print('$nama (Programmer) sedang minum kopi');
  }
}

// Class Dokter
class Dokter implements Pekerja {
  String nama;
  Dokter(this.nama);

  @override
  void bekerja() {
    print('$nama (Dokter) sedang memeriksa pasien');
  }

  @override
  void istirahat() {
    print('$nama (Dokter) sedang istirahat di ruang dokter');
  }
}

// Class Guru
class Guru implements Pekerja {
  String nama;
  Guru(this.nama);

  @override
  void bekerja() {
    print('$nama (Guru) sedang mengajar di kelas');
  }

  @override
  void istirahat() {
    print('$nama (Guru) sedang memeriksa tugas');
  }
}

// Demonstrasi polimorfisme dengan List
void main() {
  List daftarPekerja = [
    Programmer('Andi'),
    Dokter('Budi'),
    Guru('Citra'),

  print('=== Demonstrasi Polimorfisme Pekerja ===');

Bonus 3

Kode ini mendemonstrasikan Static Members - properti dan method yang dimiliki oleh class itu sendiri, bukan oleh instance/objek class. Static member diakses langsung melalui nama class, tanpa perlu membuat objek.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

import 'dart:math';

class MathUtils {
  // 1. Factorial
  static int factorial(int n) {
    if (n < 0) throw ArgumentError('Tidak bisa faktorial bilangan negatif');
    if (n == 0 || n == 1) return 1;
    int hasil = 1;
    for (int i = 2; i <= n; i++) {
      hasil *= i;
    }
    return hasil;
  }

  // 2. Cek bilangan prima
  static bool isPrima(int n) {
    if (n <= 1) return false;
    if (n == 2) return true;
    if (n % 2 == 0) return false;
    for (int i = 3; i <= sqrt(n); i += 2) {
      if (n % i == 0) return false;
    }
    return true;
  }

  // 3. Membulatkan double ke desimal tertentu
  static double roundToDecimal(double value, int decimalPlaces) {
    double faktor = pow(10, decimalPlaces).toDouble();
    return (value * faktor).round() / faktor;
  }

  // 4. Menghasilkan bilangan acak integer dalam range [min, max]
  static int randomInt(int min, int max) {
    final random = Random();
    return min + random.nextInt(max - min + 1);
  }

  // 5. Menghasilkan bilangan acak double dalam range [min, max]
  static double randomDouble(double min, double max) {
    final random = Random();
    return min + random.nextDouble() * (max - min);
  }

  // 6. Menghitung rata-rata dari list angka
  static double average(List angka) {
    if (angka.isEmpty) return 0.0;
    num sum = 0;
    for (var a in angka) {
      sum += a;
    }
    return sum / angka.length;
  }

  // 7. Menghitung median dari list angka
  static double median(List angka) {
    if (angka.isEmpty) return 0.0;
    List sorted = List.from(angka)..sort();
    int tengah = sorted.length ~/ 2;
    if (sorted.length % 2 == 0) {
      return (sorted[tengah - 1] + sorted[tengah]) / 2;
    } else {
      return sorted[tengah].toDouble();
    }
  }
}

void main() {
  print('=== MathUtils Static Methods ===');
  print('Factorial 5: ${MathUtils.factorial(5)}');
  print('Apakah 17 prima? ${MathUtils.isPrima(17)}');
  print('Apakah 20 prima? ${MathUtils.isPrima(20)}');
  print('Round 3.14159 ke 2 desimal: ${MathUtils.roundToDecimal(3.14159, 2)}');
  print('Random int (1-10): ${MathUtils.randomInt(1, 10)}');
  print('Random double (0-1): ${MathUtils.randomDouble(0, 1)}');
  print('Average [1,2,3,4,5]: ${MathUtils.average([1, 2, 3, 4, 5])}');
  print('Median [1,2,3,4,5]: ${MathUtils.median([1, 2, 3, 4, 5])}');
  print('Median [1,2,3,4,5,6]: ${MathUtils.median([1, 2, 3, 4, 5, 6])}');
}

Challenge

Sistem ini menggabungkan Abstract Class dan Interface Class untuk membuat sistem media player yang fleksibel. Abstract class MediaPlayer memberikan identitas dasar, sementara interface Playable dan Pausable memberikan kontrak perilaku.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

// Abstract class
abstract class MediaPlayer {
  String get namaMedia;
  void info();
}

// Interface Playable
abstract class Playable {
  void play();
  void stop();
}

// Interface Pausable
abstract class Pausable {
  void pause();
  void resume();
}

// AudioPlayer
class AudioPlayer implements MediaPlayer, Playable, Pausable {
  String judulLagu;
  String artis;

  AudioPlayer(this.judulLagu, this.artis);

  @override
  String get namaMedia => 'Audio Player';

  @override
  void info() {
    print('Audio Player: $judulLagu oleh $artis');
  }

  @override
  void play() {
    print('Memutar lagu: $judulLagu');
  }

  @override
  void stop() {
    print('Menghentikan lagu: $judulLagu');
  }

  @override
  void pause() {
    print('Menjeda lagu: $judulLagu');
  }

  @override
  void resume() {
    print('Melanjutkan lagu: $judulLagu');
  }
}

// VideoPlayer
class VideoPlayer implements MediaPlayer, Playable, Pausable {
  String judulVideo;
  String durasi;

  VideoPlayer(this.judulVideo, this.durasi);

  @override
  String get namaMedia => 'Video Player';

  @override
  void info() {
    print('Video Player: $judulVideo (Durasi: $durasi)');
  }

  @override
  void play() {
    print('Memutar video: $judulVideo');
  }

  @override
  void stop() {
    print('Menghentikan video: $judulVideo');
  }

  @override
  void pause() {
    print('Menjeda video: $judulVideo');
  }

  @override
  void resume() {
    print('Melanjutkan video: $judulVideo');
  }
}

// Main
void main() {
  print('=== Media Player System ===\n');

  List mediaList = [
    AudioPlayer('Bohemian Rhapsody', 'Queen'),
    VideoPlayer('Inception', '2 jam 28 menit'),
  ];

  for (var media in mediaList) {
    media.info();

    if (media is Playable) {
      (media as Playable).play();
      (media as Playable).stop();
    }

    if (media is Pausable) {
      (media as Pausable).pause();
      (media as Pausable).resume();
    }

    print('----------------');
  }
}
