pengertian

Multiple inheritance adalah konsep di mana sebuah class dapat mewarisi sifat (atribut & method) dari lebih dari satu class induk (parent class).

Class modifiers adalah kata kunci yang digunakan untuk mengubah perilaku atau sifat dari sebuah class, seperti `abstract`, `interface`, `sealed`, dan `base`.

Studi Kasus: mixin dengan state

Mixin digunakan untuk menambahkan kemampuan ke dalam sebuah class tanpa harus menggunakan banyak pewarisan.Contohnya, class Bird bisa terbang, sedangkan Duck bisa terbang dan juga berenang karena menggunakan dua mixin yang berbeda.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

mixin CanFly {
  bool _isFlying = false;
  bool get isFlying => _isFlying;

  void fly() {
    if (!_isFlying) {
      _isFlying = true;
      print('Started flying!');
    } else {
      print('Already flying!');
    }
  }

  void land() {
    if (_isFlying) {
      _isFlying = false;
      print('Landed!');
    } else {
      print('Already on the ground!');
    }
  }
}

mixin CanSwim {
  bool _isSwimming = false;
  bool get isSwimming => _isSwimming;

  void swim() {
    if (!_isSwimming) {
      _isSwimming = true;
      print('Started swimming!');
    } else {
      print('Already swimming!');
    }
  }

  void stopSwim() {
    if (_isSwimming) {
      _isSwimming = false;
      print('Stop swimming!');
    } else {
      print('Already on stop swimming!');
    }
  }
}

class Bird with CanFly {}
class Duck extends Bird with CanSwim {}
class Fish with CanSwim {}
class Airplane with CanFly {}

void main() {
  Bird bird = Bird();
  bird.fly();
  bird.fly();
  bird.land();
  bird.land();

  Duck duck = Duck();
  duck.fly();
  duck.swim();
  duck.swim();
  duck.stopSwim();
}

Sealed Class

Sealed class adalah class yang membatasi dan “mengunci” semua kemungkinan turunannya, sehingga hanya class tertentu saja yang boleh mewarisinya.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

sealed class ApiResponse {}

class Success extends ApiResponse {
  final String message;
  Success(this.message);
}

class Error extends ApiResponse {
  final String errorMessage;
  Error(this.errorMessage);
}

void handleResponse(ApiResponse response) {
  if (response is Success) {
    print("Success: ${response.message}");
  } else if (response is Error) {
    print("Error: ${response.errorMessage}");
  }
}

void main() {
  handleResponse(Success("Berhasil"));
  handleResponse(Error("Gagal"));
}

base class

Base class adalah class yang mengatur cara pewarisan Base class itu dibuat supaya class lain hanya boleh menggunakan dia dengan cara tertentu, yaitu: Boleh di-extend (extends) Artinya class baru benar-benar mewarisi semua sifat dan perilaku dari base class Tidak boleh di-implement (implements)

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

base class Animal {
  void speak() {}
}

base class Dog extends Animal {
  @override
  void speak() {
    print("Woof!");
  }
}

base class Cat extends Animal {
  @override
  void speak() {
    print("Meow!");
  }
}

void main() {
  Animal dog = Dog();
  dog.speak();

  Animal cat = Cat();
  cat.speak();
}

final class

Final class adalah class yang tidak bisa di-extend (diturunkan) oleh class lain. Keyword `final` digunakan untuk mendeklarasikan class yang tidak bisa diturunkan.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

final class Settings {
  final String theme;
  final int fontSize;

  Settings({required this.theme, required this.fontSize});

  void showSettings() {
    print("Theme: $theme, Font Size: $fontSize");
  }
}

base class CustomSettings extends Settings {
  CustomSettings(String theme, int fontSize)
      : super(theme: theme, fontSize: fontSize);
}

void main() {
  Settings settings = Settings(theme: "Dark", fontSize: 14);
  settings.showSettings();

  CustomSettings custom = CustomSettings("Light", 16);
  custom.showSettings();
}

Praktik 1 : Sealed Class dengan Pattern Matching

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

sealed class Status {
  const Status();
}

class Loading extends Status {
  const Loading();
}

class Loaded extends Status {
  final String data;
  const Loaded(this.data);
}

class Failed extends Status {
  final String error;
  const Failed(this.error);
}

void handleStatus(Status status) {
  switch (status) {
    case Loading():
      print('Loading...');
    case Loaded(:final data):
      print('Data: $data');
    case Failed(:final error):
      print('Error: $error');
  }
}

void main() {
  handleStatus(Loading());
  handleStatus(Loaded("Data masuk"));
  handleStatus(Failed("Error"));
}

Studi Kasus : Sistem State Management

Sistem State Management adalah cara mengatur dan mengontrol kondisi (state) yang terjadi dalam sebuah aplikasi selama program berjalan.State di sini maksudnya adalah keadaan aplikasi pada saat tertentu.

contoh
// Nama   : Agis Lyoni Putri
// Nim    : 251410022
// Kelas  : SI2A

sealed class AppState {}

class LoadingState extends AppState {}

class SuccessState extends AppState {
  final List items;
  SuccessState(this.items);
}

class ErrorState extends AppState {
  final String message;
  ErrorState(this.message);
}

void updateUI(AppState state) {
  switch (state) {
    case LoadingState():
      print('⏳ Loading...');
    case SuccessState(:final items):
      print('✅ Success!');
      for (int i = 0; i < items.length; i++) {
        print('Item ${i + 1}: ${items[i]}');
      }
    case ErrorState(:final message):
      print('❌ Error: $message');
  }
}

void main() {
  updateUI(LoadingState());
  updateUI(SuccessState(['A', 'B', 'C']));
  updateUI(ErrorState("Gagal"));
}
