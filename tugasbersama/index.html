LATAR BELAKANG

Zakat merupakan salah satu kewajiban umat Islam yang harus dikelola dengan baik agar dapat disalurkan kepada orang yang berhak menerimanya. Namun, masih banyak masjid yang melakukan pendataan zakat secara manual sehingga sering terjadi kesalahan pencatatan data, kesulitan menghitung jumlah zakat yang terkumpul, dan lambatnya proses pembuatan laporan.

Oleh karena itu, diperlukan sebuah sistem yang dapat membantu pengelolaan zakat secara lebih terstruktur dengan menggunakan konsep Object Oriented Programming (OOP).

2. Mengapa Masalah Ini Lebih Baik Diselesaikan dengan OOP?

Masalah ini cocok diselesaikan menggunakan OOP karena terdapat beberapa objek yang saling berhubungan dalam sistem pengelolaan zakat, yaitu Muzakki, Mustahik, Zakat, dan Amil.

Objek yang Terlibat

Muzakki

Atribut:

  • nama
  • alamat
  • jumlahZakat

Method:

  • bayarZakat()

Mustahik

Atribut:

  • nama
  • kategori

Method:

  • terimaZakat()

Zakat

Atribut:

  • jenisZakat
  • jumlah

Method:

  • tampilkanInfo()

Amil

Atribut:

  • namaAmil

Method:

  • distribusiZakat()

Dengan OOP, setiap objek memiliki data dan tugasnya masing-masing sehingga program menjadi lebih terstruktur, mudah dipahami, dan mudah dikembangkan.

3. Langkah-Langkah Menyelesaikan Masalah

Langkah 1

Mengidentifikasi permasalahan yang terjadi dalam pengelolaan zakat, seperti kesalahan pencatatan data, kesulitan menghitung total zakat, dan lambatnya pembuatan laporan.

Langkah 2

Menentukan objek yang terlibat dalam sistem, yaitu Muzakki, Mustahik, Zakat, dan Amil.

Langkah 3

Membuat class untuk setiap objek beserta atribut dan method yang sesuai dengan fungsinya.

Langkah 4

Membuat hubungan antar objek sehingga proses pembayaran, pencatatan, dan distribusi zakat dapat dilakukan secara terstruktur.

Langkah 5

Mengimplementasikan program menggunakan konsep Object Oriented Programming (OOP) untuk mengelola data zakat secara lebih efektif dan efisien.

class Orang {
  String nama;
  String alamat;

  Orang(this.nama, this.alamat);

  void tampilkanInfo() {
    print("Nama : $nama");
    print("Alamat : $alamat");
  }
}

class Muzakki extends Orang {
  double jumlahZakat;

  Muzakki(
    String nama,
    String alamat,
    this.jumlahZakat,
  ) : super(nama, alamat);

  void bayarZakat() {
    print("$nama membayar zakat sebesar Rp$jumlahZakat");
  }
}

class Mustahik extends Orang {
  String kategori;

  Mustahik(
    String nama,
    String alamat,
    this.kategori,
  ) : super(nama, alamat);

  void terimaZakat(double jumlah) {
    print("$nama menerima zakat sebesar Rp$jumlah");
  }
}

class Zakat {
  String jenisZakat;
  double jumlah;

  Zakat(this.jenisZakat, this.jumlah);

  void tampilkanInfo() {
    print("Jenis Zakat : $jenisZakat");
    print("Jumlah : Rp$jumlah");
  }
}

class Amil {
  String namaAmil;

  Amil(this.namaAmil);

  void distribusiZakat(
      Mustahik mustahik,
      double jumlah) {
    print(
        "$namaAmil menyalurkan zakat kepada ${mustahik.nama}");
    mustahik.terimaZakat(jumlah);
  }
}

void main() {
  Muzakki muzakki =
      Muzakki("Ahmad", "Palembang", 500000);

  Mustahik mustahik =
      Mustahik("Fatimah", "Palembang", "Fakir");

  Zakat zakat =
      Zakat("Zakat Fitrah", 500000);

  Amil amil =
      Amil("Uliya");

  muzakki.tampilkanInfo();
  muzakki.bayarZakat();

  zakat.tampilkanInfo();

  amil.distribusiZakat(
      mustahik,
      zakat.jumlah);
}
