Latar Belakang

Zakat merupakan salah satu kewajiban umat Islam yang harus dikelola dengan baik agar dapat disalurkan kepada orang yang berhak menerimanya. Namun, masih banyak masjid yang melakukan pendataan zakat secara manual sehingga sering terjadi kesalahan pencatatan data, kesulitan menghitung jumlah zakat yang terkumpul, dan lambatnya proses pembuatan laporan.

Oleh karena itu, diperlukan sebuah sistem yang dapat membantu pengelolaan zakat secara lebih terstruktur dengan menggunakan konsep Object Oriented Programming (OOP).

Permasalahan

  • Kesalahan pencatatan data muzakki dan mustahik.
  • Sulit menghitung total zakat yang terkumpul.
  • Distribusi zakat kurang terorganisir.
  • Pembuatan laporan membutuhkan waktu yang lama.
  • Data mudah hilang atau rusak.

Mengapa Menggunakan OOP?

Sistem Pengelolaan Zakat Masjid menggunakan OOP karena terdapat beberapa objek yang saling berhubungan, yaitu Orang, Muzakki, Mustahik, Zakat, dan Amil.

Dengan OOP, setiap objek memiliki atribut dan method masing-masing sehingga program menjadi lebih terstruktur, mudah dipahami, mudah dikembangkan, serta mengurangi pengulangan kode melalui konsep inheritance.

Orang merupakan parent class. Muzakki dan Mustahik merupakan child class yang mewarisi atribut dan method dari class Orang.

Hubungan Class dan Object

Class merupakan cetak biru (blueprint), sedangkan object merupakan hasil pembuatan dari class.

  • Class Orang → Object Orang
  • Class Muzakki → Object muzakki
  • Class Mustahik → Object mustahik
  • Class Zakat → Object zakat
  • Class Amil → Object amil

Class Zakat

Source Code 1

merupakan class Orang yang berfungsi sebagai parent class atau kelas induk. Class ini digunakan untuk menyimpan data umum yang dimiliki oleh setiap orang dalam sistem, yaitu nama dan alamat. Selain itu, class ini memiliki method tampilkanInfo() yang digunakan untuk menampilkan informasi data orang. Dengan adanya class induk, penulisan program menjadi lebih efisien karena data yang sama tidak perlu ditulis berulang kali pada class lain.

DART SOURCE CODE


// Nama   : Uliya Gumpari (251410008)
// Nama   : Agis Lyoni Putri (251410022)

// ===================================================
// CLASS ORANG
// Class induk (parent class) yang menyimpan
// data umum berupa nama dan alamat
// ===================================================
class Orang {

  // Properti untuk menyimpan nama orang
  String nama;

  // Properti untuk menyimpan alamat orang
  String alamat;

  // Constructor untuk mengisi nilai nama dan alamat
  Orang(this.nama, this.alamat);

  // Method untuk menampilkan informasi orang
  void tampilkanInfo() {
    print("Nama   : $nama");
    print("Alamat : $alamat");
  }
}

// ===================================================
// CLASS MUZAKKI
// Class turunan dari Orang yang digunakan
// untuk menyimpan data pembayar zakat
// ===================================================
class Muzakki extends Orang {

  // Properti jumlah zakat yang dibayarkan
  double jumlahZakat;

  // Constructor
  Muzakki(
    String nama,
    String alamat,
    this.jumlahZakat,
  ) : super(nama, alamat);

  // Method untuk proses pembayaran zakat
  void bayarZakat() {
    print("$nama telah membayar zakat sebesar Rp$jumlahZakat");
  }
}

// ===================================================
// CLASS MUSTAHIK
// Class turunan dari Orang yang digunakan
// untuk menyimpan data penerima zakat
// ===================================================
class Mustahik extends Orang {

  // Properti kategori penerima zakat
  String kategori;

  // Constructor
  Mustahik(
    String nama,
    String alamat,
    this.kategori,
  ) : super(nama, alamat);

  // Method untuk menerima zakat
  void terimaZakat(double jumlah) {
    print("$nama menerima bantuan zakat sebesar Rp$jumlah");
  }
}

// ===================================================
// CLASS ZAKAT
// Digunakan untuk menyimpan informasi zakat
// ===================================================
class Zakat {

  // Properti jenis zakat
  String jenisZakat;

  // Properti jumlah zakat
  double jumlah;

  // Constructor
  Zakat(this.jenisZakat, this.jumlah);

  // Method untuk menampilkan data zakat
  void tampilkanInfo() {
    print("Jenis Zakat : $jenisZakat");
    print("Jumlah      : Rp$jumlah");
  }
}

// ===================================================
// CLASS AMIL
// Digunakan untuk mengelola dan menyalurkan zakat
// ===================================================
class Amil {

  // Properti nama amil
  String namaAmil;

  // Constructor
  Amil(this.namaAmil);

  // Method untuk menyalurkan zakat kepada mustahik
  void distribusiZakat(
      Mustahik mustahik,
      double jumlah) {

    print(
      "$namaAmil menyalurkan zakat kepada ${mustahik.nama}");

    mustahik.terimaZakat(jumlah);
  }
}

// ===================================================
// PROGRAM UTAMA
// Menghubungkan seluruh class dalam satu sistem
// ===================================================
void main() {

  // Membuat objek Muzakki
  Muzakki muzakki =
      Muzakki(
        "Bapak H. Abdullah",
        "Palembang",
        750000,
      );

  // Membuat objek Mustahik
  Mustahik mustahik =
      Mustahik(
        "Ibu Siti Aminah",
        "Palembang",
        "Fakir",
      );

  // Membuat objek Zakat
  Zakat zakat =
      Zakat(
        "Zakat Fitrah",
        750000,
      );

  // Membuat objek Amil
  Amil amil =
      Amil(
        "Uliya dan Agis",
      );

  print("=== SISTEM PENGELOLAAN ZAKAT MASJID ===\n");

  // Menampilkan data muzakki
  print("DATA MUZAKKI");
  muzakki.tampilkanInfo();
  muzakki.bayarZakat();

  // Menampilkan data zakat
  print("\nDATA ZAKAT");
  zakat.tampilkanInfo();

  // Menyalurkan zakat kepada mustahik
  print("\nDISTRIBUSI ZAKAT");
  amil.distribusiZakat(
    mustahik,
    zakat.jumlah,
  );
}
